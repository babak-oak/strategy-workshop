Blue Ocean Strategy Workbench

01

Existing value proposition vs peers

Score every competing factor 0–10 for your company and each peer. The strategy canvas draws the value curves; classify each factor in the ERRC grid below.

Rows = competing factors of the market. Columns = players. Your company curve is drawn in ocean blue; peers in red-ocean tones.

ERRC grid

Assign each factor: Eliminate, Reduce, Raise or Create. This drives the to-be move and the barriers register in step 02.

02

To-be value proposition & barriers to imitation

Set the target (to-be) level for each factor. For every Raise and Create factor, register the barriers that protect your lead and rate each barrier's strength.

The dashed line is your to-be curve; the solid line your current curve; the muted line is the peer average.

Barriers-to-imitation register

Only Raise / Create factors appear here — these are the lead positions worth defending. Add the moats that make imitation hard and rate strength 1–5.

04

Portfolio map — future (company only)

Set the target position for each of your products. Arrows show the intended migration from today toward the pioneer zone.

Hollow bubbles = today, solid bubbles = target. The arrow is the migration you commit to. Edit target level, share and revenue per product.

05

Value deployment — focus the organisation

Tie each Raise/Create value factor to the parts of the company that must deliver it. Choose a simple deployment matrix or a full Hoshin Kanri X-matrix.

Rows = organisational units, columns = Raise/Create value factors. In each cell name the owner, the objective and the KPI. Empty cells mean an undefended value factor.

07

Product-owner focus by stage

For each stage of the product journey, set where the product owner should focus, the objectives, the key activities and the exit KPI.

Idea refinement → Idea development → Deployment → Go-to-market. Keep each stage's focus tight so owners know what "done" looks like before moving on.

Next steps

List the key results you want to reach in the next 1–3 months, assign each to an existing or prospective team member, then push them to a weekly Google Calendar review.
